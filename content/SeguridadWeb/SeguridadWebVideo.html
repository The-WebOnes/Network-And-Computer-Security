¿Cómo se relaciona SSL/TLS con HTTPS?

Cuando se instala un certificado, se configuras para transmitir datos utilizando HTTPS. Las dos tecnologías van de la mano y no se puede usar una sin la otra. Las URL están precedidas con HTTP (Protocolo de transferencia de hipertexto) o HTTPS (Protocolo de transferencia de hipertexto seguro). Esto es efectivamente lo que determina cómo se transmiten los datos que se envían y recibe, una de las grandes diferencias entre http y https. .
